Akemi
==========
scala kestrel and redis sample


Usage:
--------
       -c     conf check mode

       -p     conf file path

       --help display this help and exit

To run 
--------
    sbt 'run -p conf/AkemiConfigSample.scala'


Test
-------- 
    sbt test
